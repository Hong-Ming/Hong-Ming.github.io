Terminal Configuration
Home/Posts/Terminal Configuration

This post is to keep my terminal configuration files and information so that I can quickly rebuild it in other machine. The fowllowing picture is a snapshop of my termainl, feel free to use my terminal configuration.

Prerequisites
  • Shell: Zsh
  • Framework: On My Zsh
  • Package Manager: Homebrew
  • Plugin Manager: Antibody
  • Fonts: Nerd Fonts
  • Packages:
    • colorls
    • bat
    • trash-CLI
  • Plugins:
    • zsh-syntax-highlighting
    • zsh-autosuggestions
    • zsh-completions
    • powerlevel9k
    • git
    • colored-man-pages
    • autojump
    • pip
